FormFooter(doubleLinkopt, topLinkopt, bottomLinkopt, tooltipopt)

Description:

  • The footer for the auth forms. Includes links to other pages, back button, and info tooltips. All params are optional but at least one link must be provided.

Source:
  • components/auth/FormFooter.js, line 59
Examples
// doubleLink object
const footerData = {
      label: " Having trouble logging in?",
      firstLink: {
        to: "/userRecovery/forgotUsername",
        text: "Forgot Username"
      },
      secondLink: {
        to: "/userRecovery/forgotPassword",
        text: "Forgot Password"
      }
    }
    <FormFooter doubleLink={footerData} />
// topLink object with label
const footerData = {
      to: "/register",
      text: "Create an Account",
      label: "Need an account?"
    }
   <FormFooter topLink={footerData} />
// topLink and bottomLink objects
const footerData = {
      topLink: {
        to: "/userRecovery/forgotUsername",
        text: "Forgot Username"
      },
      bottomLink: {
        to: "/login",
        text: "Cancel",
      }
  }
  <FormFooter topLink={footerData.topLink} bottomLink={footerData.bottomLink} />
Parameters:
Name Type Attributes Description

the object used to render the first link in the double link. The object has the same structure as the topLink and bottomLink objects.

the object used to render the second link in the double link The object has the same structure as the topLink and bottomLink objects.
topLink object <optional>

the object used to render a single navigation link. Renders below the double link but above the bottom link
